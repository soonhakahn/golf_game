# 3D Premium Golf Game

GitHub에서 바로 실행 가능한 3D 골프 시뮬레이션 샘플입니다.

## 실행

1. 저장소 루트에서 `index.html` 더블클릭 또는 로컬 서버로 실행
   - 로컬 서버 예시: `python -m http.server 5173`
2. 브라우저에서 `http://localhost:5173` 열기

## 조작법

- `A/D`: 조준 방향
- `W/S`: 클럽 로프트
- `Space` 또는 마우스 왼쪽 버튼 누름: 파워 게이지 충전
- 마우스 버튼/스페이스 뗌: 스윙
- `1~4`: 클럽 선택
- `C`: 카메라 모드 전환
- `R`: 공 리셋

## 핵심 기능

- 실제감 있는 3D 코스 + 조명/그림자
- 4개 클럽(드라이버/아이언/웨지/퍼터): 거리/정확도/스핀 반영
- 중력, 스핀, 바운스, 경사면 롤링 물리
- 바람, 바람 방향 UI, 습기/모래/잔디 이펙트
- 카메라 모드(조준/스윙/팔로우/리플레이)
- 모드: Practice / Tournament / Challenge
